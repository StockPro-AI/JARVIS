# Jarvis - Python Voice Assistant 🤖

## Features
- Voice command recognition
- Opens websites
- Plays music
- Fetches news using API
- AI responses using Gemini

## Tech Stack
- Python
- SpeechRecognition
- pyttsx3
- Requests
- Google Gemini API

## Setup

1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt

3. Create a `.env` file:
   NEWS_API_KEY=your_key
   GEMINI_API_KEY=your_key

4. Run:
   python main.py
